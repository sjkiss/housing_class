There are 2872 respondents in the module. There are three variables to consider.

  1. occupation_code is the 5-digit NOC code respondents selected in the menu that we provided. It was offered to all respondents, retired, student or not. This variable is 5-digits long if respondents selected a code. It is stored as a character variable, so if respondents picked nothing, then it is just empty.
  2. pes_occ_select_2 is the open-ended text response that was offered to respondents at the end of the menu selection process if they could not see their response. This variable has the response -99 if it was seen but not selected, sometimes it is blank, and many occasions a text response such as retired or yes or actual occupations. We coded those occupations as best we could.
  3. NOC21_5. This variable is the combination either of occupation_code from the survey menu or the coding process by the RAs.
